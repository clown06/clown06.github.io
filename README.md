# HealthScan - 基于骨架的健康检测（先验知识增强的健康数据分析）

## 项目简介

健康数据分析问题至关重要，本文提出了先验知识增强的骨架动作识别框架，框架提供阿尔兹海默症检测、跌倒检测等健康监测服务，详细介绍了该框架背后的技术原理，促进健康数据分析领域的交流与合作。

## 功能特点

- 响应式设计，适配各种设备屏幕
- 现代化UI界面，用户体验友好
- 展示基于骨架的动作识别技术
- 提供健康检测服务介绍
- 包含联系表单功能
